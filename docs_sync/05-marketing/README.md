# Marketing

Camada de entrada no mercado, monetização inicial e preparação comercial da DIGIAI.

## Regra atual
No estágio atual, o Clearix Academy opera em lógica `marketplace-first`.

Isso significa:
- aquisição principal via `Hotmart` e `Kiwify`
- produto de entrada vendido como `ebook / manual visual em PDF`
- entrega complementar de `90 dias na Nexus`
- objetivo estratégico de gerar caixa inicial, base de compradores e ponte futura para o `Clearix`
