# Prompt da landing de Ótica Sem Improviso

## Classificação documental
- tipo: prompt operacional de criação
- status: canônico atual
- última revisão: 2026-04-23
- responsável formal: Precisa de validação manual

## Objetivo do documento
Registrar o prompt base para criar ou iterar a landing externa de apoio do produto Ótica Sem Improviso.

## Regra de uso
Este prompt não deve transformar a landing em motor principal de aquisição.

Ele existe para:
- manter coerência com o produto
- sustentar tráfego externo quando houver
- encaminhar com clareza para Hotmart ou Kiwify

## Prompt oficial

```text
Crie ou refine uma landing externa de apoio para o produto Ótica Sem Improviso.

Verdade da oferta:
- Produto principal: Ótica Sem Improviso
- Subtítulo: Manual Visual de Atendimento e Conversão para Óticas
- Formato comercial: ebook / manual visual em PDF
- Complemento: 90 dias de acesso na Nexus
- Canal principal de descoberta hoje: marketplaces como Hotmart e Kiwify
- Papel da landing: apoiar conversão de tráfego externo, não substituir o marketplace

Público principal:
- vendedor de ótica
- atendente
- consultor óptico
- colaborador de linha de frente

Promessa central:
Em 3 dias, você sai do atendimento no improviso, responde melhor no WhatsApp e aprende a vender com mais segurança sem depender de desconto.

Regras:
- vender primeiro o ebook
- tratar a Nexus como complemento
- falar com quem vive balcão real
- manter o Clearix apenas como referência sutil de ecossistema
- evitar tom de curso genérico
- evitar urgência artificial
- evitar promessa inflada
- usar CTA para checkout do marketplace

Estrutura sugerida:
1. Hero
2. Problema
3. Transformação
4. O que é
5. O que vem dentro
6. Módulos
7. Nexus como apoio
8. Para quem é / não é
9. Oferta
10. FAQ
11. CTA final
```
