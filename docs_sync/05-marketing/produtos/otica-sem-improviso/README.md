# Otica Sem Improviso

Pasta de documentacao viva do OSI dentro do DIGIAI.

Posicionamento:
- OSI e a primeira isca paga da esteira de guias para o varejo optico.
- O guia precisa vender e entregar valor real, mas nao e o produto final.
- O produto final e o ecossistema de apps para oticas.
- A funcao do OSI e comprar atencao qualificada, educar o mercado e preparar a ascensao.

Regra central:

> O guia resolve uma parte da dor. O ecossistema resolve a rotina.

Subpastas:
- `01-estrategia`: arquitetura, tese de ecossistema, mapa de guias e ascensao.
- `04-operacao-e-venda`: funil, checkout, ROAS, trafego e recuperacao.
- `05-copys-e-prompts`: banco completo de copys, prompts e JSONs para agent visual.

Arquivos principais:
- `01-estrategia/arquitetura-ecossistema-apps-oticas.md`
- `04-operacao-e-venda/plano-configuracao-funil-osi.md`
- `04-operacao-e-venda/regua-recuperacao-carrinho-osi.md`
- `04-operacao-e-venda/projecao-roas-osi.xlsx`
- `05-copys-e-prompts/prompt_mestre_copy_osi.md` (prompt para gerar pecas)
- `05-copys-e-prompts/indice_copys.json` (mapa de todos os arquivos)
