# Copys e Prompts — Ótica Sem Improviso

Pasta centralizada com todas as copys, prompts e diretrizes criativas do OSI.

## Arquivos

### Prompts (para agent visual)
- `prompt_mestre_copy_osi.md` — Prompt completo para gerar/revisar qualquer peça de copy do OSI
- `prompt_criativos_ads.md` — Prompt específico para criar anúncios

### JSONs de Copy (consumíveis por agents)
- `copy_landing_page.json` — Landing page completa estruturada
- `copy_criativos_ads.json` — 8 criativos para Meta Ads
- `copy_email_sequences.json` — 10 e-mails (captação + onboarding + abandono)
- `copy_stories_carrossel.json` — Carrosséis + Stories Instagram
- `copy_video_scripts.json` — Roteiros de vídeo (VSL + Reels)
- `copy_whatsapp_bio.json` — WhatsApp broadcast + Bio IG
- `copy_pagina_obrigado.json` — Thank You Page
- `copy_design_system.json` — Design system (cores, fontes, componentes)

### Índice
- `indice_copys.json` — Mapa mestre de todos os arquivos

## Como usar

1. O agent visual consome os JSONs diretamente
2. O `copy_design_system.json` define regras visuais (sempre consultar primeiro)
3. O `prompt_mestre_copy_osi.md` serve para regenerar/adaptar qualquer peça
4. Novos formatos devem seguir o mesmo padrão JSON
