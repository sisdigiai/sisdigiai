- separar documentação viva por produto
- concentrar insumos, versões, decisões e estrutura de entrega
- evitar que os docs estratégicos fiquem inchados

- documentos canônicos continuam em `05-marketing`
- cada produto pode ter sua própria pasta de apoio
- insumo bruto entra primeiro na pasta do produto e depois sobe para os docs canônicos quando fizer sentido
