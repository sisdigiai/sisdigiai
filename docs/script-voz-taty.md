# 🎙️ Script de gravação — Voz Taty (ElevenLabs Voice Cloning)

> **Pra quê:** clonar a voz da Taty no ElevenLabs pra usar em TODOS os reels OSI. Vendedor vai escutar SEMPRE a Taty falando — vira marca sonora.

---

## ⚙️ Como gravar

| Item | Como fazer |
|---|---|
| **Lugar** | Sala silenciosa (banheiro fechado funciona surpreendentemente bem) |
| **Equipamento** | Celular bom (iPhone 12+/Galaxy S20+) **com fone com microfone** (Apple, JBL, Galaxy) |
| **Posição** | Microfone do fone a ~10cm da boca |
| **App** | Gravador padrão do celular (Voice Memos iOS / Gravador Samsung / Google Recorder) |
| **Formato** | MP3 ou M4A (qualidade automática do app já basta) |
| **Take único** | Lê tudo de uma vez, sem cortar. Se errar, segue. Falhas naturais ajudam o cloning. |

**Tempo total:** ~3 minutos lendo o script abaixo.

## 📜 SCRIPT — leia natural, como se estivesse conversando

### Bloco 1 — Apresentação (tom neutro, calmo)

```
Oi, eu sou a Taty.
Tenho mais de quarenta anos atrás do balcão de ótica.
Vou te contar algumas coisas que aprendi atendendo cliente todo dia.
```

*[pausa 2 segundos]*

### Bloco 2 — Conversado (tom de explicação, mais animado)

```
Quando o cliente entra perguntando "quanto custa",
ele não tá pedindo preço. Tá pedindo segurança.

Se você responder o preço na hora, perdeu a venda
antes de começar.
```

*[pausa 2 segundos]*

### Bloco 3 — Sério (afirmando, voz firme)

```
O método dos cinco movimentos não é teoria.
É o que eu aplico no balcão todo dia há quarenta anos.
Funciona porque é prática, não invenção.
```

*[pausa 2 segundos]*

### Bloco 4 — Animado (convite)

```
Salva esse vídeo.
Se você atende ótica e quer parar de improvisar, vai te servir.
Manda também pro colega que tá com a meta apertada esse mês.
```

*[pausa 2 segundos]*

### Bloco 5 — Calmo (encerrando)

```
Ótica sem improviso.
Vendedor que prepara, vende mais.
É isso. Até a próxima.
```

*[pausa 3 segundos]*

### Bloco 6 — Perguntas (variando entonação ao final de cada frase)

```
Já aconteceu de você perder a venda no "tá caro"?

Quantas vezes o cliente saiu sem fechar?

O que muda quando você pergunta o uso antes do preço?
```

*[pausa 2 segundos]*

### Bloco 7 — Frases curtas (pausa de 1s entre cada uma)

```
Atende com método.

Pergunta antes de mostrar.

Sustenta o valor.

Recupera no WhatsApp.

Fecha sem desconto.
```

## 🤖 Clonagem no ElevenLabs (Gilberto faz, ~5 min)

1. Acessar https://elevenlabs.io
2. Login com email OSI (criar conta free se não tiver)
3. Menu **Voice Lab** (à esquerda)
4. Botão **+ Add Voice** → **Instant Voice Cloning**
5. Configurar:
   - **Voice Name:** `Voz Taty`
   - **Description:** "Voz da fundadora da Ótica Sem Improviso. 40 anos de balcão. Tom calmo, prático, sem afetação. Brazilian Portuguese."
   - **Labels:** `female`, `mature`, `brazilian portuguese`, `warm`, `conversational`
6. Upload do arquivo `taty-voice-sample-2026-05.m4a`
7. Aceitar termos → **Add Voice**
8. Aguardar ~30 segundos
9. Voz aparece na lista. **Copiar o Voice ID** (botão `...` ao lado da voz → Copy Voice ID)
10. **Atualizar no DIGIAI app:** Cadastro Empresa → Identidade Digital → editar "Voz Taty (ElevenLabs clonada)" → colar Voice ID no campo "Valor"

## 🎬 Usar a voz clonada (cada reel daqui pra frente)

No Wizard "Produzir post":

1. **Step 3 — Voz / Narração** → escolhe "ElevenLabs voiceover Reel"
2. App gera o prompt com **as FALAS** já extraídas do roteiro
3. **Copiar prompt** → abre ElevenLabs
4. No painel ElevenLabs:
   - Voice: selecionar **"Voz Taty"**
   - Stability: 50-65%
   - Similarity boost: 75%
   - Style: 0-30%
   - Speaker boost: ON
5. **Generate** → baixar MP3
6. Voltar pro Wizard → upload do MP3 → **Salvar output**

Pronto. Voz da Taty narra o reel pra você.
